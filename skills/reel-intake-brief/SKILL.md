# Reel Intake Brief

Generate `01-creative-brief.md` from `idea-input.yaml`.

## Load these resources
- `references/segment-library.md`
- `assets/templates/01-creative-brief.template.md`

## Input
- `idea-input.yaml`

## Output
- `01-creative-brief.md`

## Required decisions
1. Define communication objective in one sentence.
2. Confirm audience and pain point.
3. Select thematic line (`economica`, `social`, `institucional`).
4. Select segment title:
   - If `segmento_preferido=auto`, rotate through segment library.
5. Produce concise promise and CTA.

## Quality checks
- Keep tone `racional_firme`.
- Avoid absolutist language.
- Do not use proper names for spokesperson role.
