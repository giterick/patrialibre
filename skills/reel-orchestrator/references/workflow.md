# Workflow Contract

## Required output files
1. `00-manifest.yaml`
2. `01-creative-brief.md`
3. `02-factcheck-matrix.md`
4. `03-script-master.md`
5. `04-shotlist-storyboard.md`
6. `05-call-sheet-lite.md`
7. `06-direccion-talento.md`
8. `07-plan-arte-y-recursos.md`
9. `08-edicion-post.md`
10. `09-publicacion-multired.md`
11. `10-checklist-pregrabacion.md`

## Parsing heuristics for `linea_tematica`
- `economica`: impuestos, empleo, emprendimiento, burocracia, inversion, mercado.
- `social`: asistencia, oportunidades, movilidad social, trabajo, comunidad.
- `institucional`: corrupcion, seguridad, estado, instituciones, justicia, orden.

## Defaults
- `segmento_preferido: auto`
- `duracion_segundos: 75`
- `plataformas: [instagram]`
- `tono: racional_firme`

## Quality gates
1. Script structure complete (hook, problema, comparacion, propuesta, cierre+cta).
2. Target duration between 60 and 90 seconds.
3. Fact-check matrix has source and date for each verifiable claim.
4. No proper names for spokesperson role.
5. Publishing includes Instagram plus TikTok/YouTube Shorts adaptation.
