# Publicacion Multired

- Formato: Reel 9:16, 75s
- Copy principal: "Cuando emprender tiene menos trabas, mas familias pueden prosperar. Comparte este video si quieres un pais de oportunidades."
- Hashtags: `#PatriaLibre #Prosperidad #Emprendimiento #RepublicaDominicana`
- CTA: "Comparte este video"
- Hora sugerida: 7:30 PM

## Adaptacion TikTok
- Ajuste de apertura: entrar directo con hook B en 2 segundos.
- Ajuste de copy: version mas corta con pregunta inicial.
- Ajuste de ritmo: cortes cada 2-3 segundos en el primer bloque.

## Adaptacion YouTube Shorts
- Titulo sugerido: "Menos trabas, mas oportunidades: como impulsar pequenos negocios"
- Descripcion sugerida: "Explicacion breve sobre impuestos, burocracia y prosperidad para pequenos negocios."
- Ajuste de cierre: agregar texto final de 2 segundos con CTA.

## Versiones de caption

- Subtitulos revisados.
